# Antigravity Starter Template

This is the starter repository for Vibecoding with Gemini and Claude. It contains the essential setup files (`CLAUDE.md`, `AGENTS.md`, `GEMINI.md`) and agent execution environments (`.agent/`, `.claude/`) to get started quickly.

## Requirements

1. **Environment:** Node.js, Git, and a code editor (like VS Code or Cursor).
2. **AI Assistants:** Claude Code or Gemini set up locally.

## Setup Instructions

1. Clone this repository directly or branch off.
2. Initialize your project (e.g., `npx -y create-next-app@latest .` if you intend to start a Next.js application).
3. Follow the data definitions and rules outlined in `GEMINI.md` and `CLAUDE.md`.
4. Run your vibecoding prompts against this context!
